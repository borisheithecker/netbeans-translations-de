Erzeugt eine Datei mit Hilfsmitteln (.properties) für internationalisierte Anwendungen durch Extraktion aller sichtbaren Texte aus dem Code. Hilfsmittel-Dateien können auch genutzt werden, um andere Arten von Texten, z.B. Eigenschaften in Ant-Skripten zu sammeln. Die erzeugte Hilfsmittel-Datei enthält nur eine Spracheinstellung, aber es können weitere Spracheinstellungen über das Kontextmenü der erzeugten Datei hinzugefügt werden. Die Hilfsmitteldatei kann als Text-Datei (Dateiformat Property-Datei) für eine spezielle Spracheinstellung bearbeitet werden oder in einer Tabelle, die Informationen für alle Sprachen anzeigt.
