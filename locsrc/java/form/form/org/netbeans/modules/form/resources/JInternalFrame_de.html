Erzeugt einen neuen internen Rahmen mit JFC (Swing) der auf einem JDesktopPane platziert werden kann, um eine MDI-Oberfläche zu erhalten. Typischerweise erzeugt man eine interne Frameklasse mit den Anzeigeelementen für jeden Frame. Jede interne Frameklasse wird wie ein JFrame behandelt. Dann eine JFrame- oder JApplet-Anwendung mit einem JDesktopPane als Oberfläche erzeugen. Als Letztes wird der interne Rahmen mit Code zum Oberflächen-Paneel hinzugefügt oder nach dem Übersetzten der internen Rahmen mit der Copy&Paste-Funktion im Projektfenster.
