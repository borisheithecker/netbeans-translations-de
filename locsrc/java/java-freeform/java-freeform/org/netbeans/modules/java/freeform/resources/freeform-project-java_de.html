Importiert eine vorhandene Java-Applikation samt Erstellungsskript in die IDE. In Projekten ohne Formvorgaben nutzen die IDE-Kommandos das übergebene Ant-Skript zum Übersetzen, Ausführen und Testen der Programme. Änderungen im Erstellungsprozess müssen direkt im Ant-Skript vorgenommen werden. Um die langfristige Projektwartung zu vereinfachen, sollte die Vorlage für "Java-Projekte mit vorhandenen Quellen" genutzt werden.
